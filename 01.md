## git이란?

1. 버젼 관리 프로그램
2. (설치됐는지 확인) 터미널에서 git --version

---
## 초기 설치시 git 세팅

1. 터미널에 아래코드를 순서대로 작성
git config --global user.name '이름'
git config --global user.email '이름@gmail.com'

---

## github에 코딩파일 '첫'업로드
1. 내 프로필 Repositories 클릭
2. New 클릭
3. Repositories name에는 코딩파일 제목
4. Create Repository 클릭

5. VS code 좌측패널에서 git 아이콘 클릭
6. Initialize Repository
7. 업로드할 파일만 +눌러서  Staged Changes로 이동
8. Message 입력칸에 어떤 것이 변경되었는지 입력하고 Commit버튼 클릭

9. 터미널에 순차적으로 복사 붙여넣기
git remote add origin https://github.com/keb221/git-study.git
 git push -u origin main

---
## github에 코딩파일 '수정 후'업로드

1. 업로드할 파일만 +눌러서  Staged Changes로 이동
2. Message 입력칸에 어떤 것이 변경되었는지 입력하고 Commit버튼 클릭
3. [Sync Changes 위 화살표] 버튼 클릭하면 내 컴퓨터에서 github로 코드 전송
